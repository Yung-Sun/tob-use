# tob-ui 

`uniapp` 的高效 `use` 库

<br />
<br />

## 特点 🐳 

1. 功能丰富 👉 60+ 的功能供你选择
2. 按需加载 👉 让你的应用体积更小
3. 灵活可配置 👉 可配置的事件过滤器
4. 支持多端 👉 支持 app，h5 和 小程序
5. composition 👉 完全由 composition-api 构建
6. vue3 👉 支持最现代化的 vue3 版本

为什么要做这个 **use 库** 呢？
